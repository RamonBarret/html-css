Tecnologia - Realidade Aumentada


De uma forma simples, Realidade Aumentada é uma tecnologia que permite que o mundo virtual seja misturado ao real, possibilitando maior interação e abrindo uma nova dimensão na maneira como realizamos as tarefas, ou mesmo a que nós incubimos nas máquinas. Assim, se você pensava que os objetos pulando para fora da tela eram elementos de filmes de ficção ciêntífica , está na hora de mudar seus conceitos. Aliás, o que acontece com a Realidade Aumentada é o contrário: você pulará para dentro do mundo virtual para interagir com objetos que só estão limitados a sua imaginação. - tecmundo.com.br/realidade-aumentada - acesso 07/2022

Acessibilidade


Acessibilidade na web significa que as pessoas com deficiência podem usar a web. Mais especificamente, a acessibilidade na web significa que estas pesssoas podem perceber, entender, navegar, interagir e contribuir para web.
